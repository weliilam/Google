# 图标文件说明

本目录需要放置以下图标文件才能正常显示扩展图标：

## 必需文件

- **icon16.png** - 16x16 像素，用于工具栏小图标
- **icon48.png** - 48x48 像素，用于扩展管理页面
- **icon128.png** - 128x128 像素，用于Chrome网上应用店

## 临时解决方案

如果暂时没有图标文件，可以：

1. 使用在线工具生成图标：
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/

2. 或使用任何PNG图片，重命名为对应文件名

3. 或使用简单的纯色图标（可以使用Paint或Photoshop创建）

## 推荐图标设计

- 使用蓝色调（与插件主题一致）
- 显示网络/数据分析相关图标（如放大镜、图表、网络节点等）
- 保持简洁、现代的设计风格

## 注意事项

- 必须是PNG格式
- 尺寸必须准确
- 背景透明效果更好

# 快速开始指南

## 🚀 5分钟快速安装

### 第一步：准备图标（可选）

如果没有图标文件，插件仍然可以安装和使用，只是不会显示自定义图标。

**快速生成图标的方法：**

1. 访问 https://www.flaticon.com/
2. 搜索 "network" 或 "analytics"
3. 下载一个喜欢的PNG图标
4. 使用在线工具调整尺寸：
   - 转换为16x16、48x48、128x128三个尺寸
   - 网址：https://www.iloveimg.com/resize-image/

或者直接跳过这一步，Chrome会使用默认图标。

### 第二步：加载扩展

1. 打开Chrome浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 打开右上角的"开发者模式"开关
4. 点击"加载已解压的扩展程序"按钮
5. 选择 `chrome-request-analyzer` 文件夹
6. ✅ 扩展加载成功！

### 第三步：开始使用

1. 打开一个有API调用的网站（推荐）：
   - https://jsonplaceholder.typicode.com/（测试API）
   - 任何你常用的网站
   
2. 点击浏览器工具栏上的扩展图标

3. 你会看到：
   - 左侧：请求列表
   - 右侧：请求详情
   - 顶部：筛选和搜索功能

## 🔧 高级技巧

### 自定义翻译字典

编辑 `popup/popup.js` 文件，找到 `translationDict` 对象：

```javascript
const translationDict = {
  // 添加你的翻译
  'customField': '自定义字段',
  'businessId': '业务ID',
  // ...
};
```

保存后重新加载扩展即可生效。

### 查看特定网站的请求

1. 访问目标网站
2. 打开扩展
3. 点击"刷新"按钮
4. 所有该网站的请求会显示出来

### 导出请求数据（需要手动添加）

在控制台中执行：

```javascript
// 获取当前请求
const requests = allRequests;
console.log(requests);

// 复制到剪贴板
copy(JSON.stringify(requests, null, 2));
```

## 🐛 常见问题

### Q: 为什么看不到任何请求？
A: 
- 确保页面已经发起了网络请求
- 点击"刷新"按钮
- 检查网页是否加载完成
- 尝试访问其他网页

### Q: 翻译不准确怎么办？
A:
- 手动编辑 `translationDict` 添加更多翻译
- 字典支持驼峰和下划线命名转换

### Q: 能否看到响应体？
A:
- 当前版本受Chrome API限制，无法获取响应体
- 未来版本会使用DevTools Protocol实现

### Q: 扩展无法加载？
A:
- 检查manifest.json语法
- 确认所有文件路径正确
- 查看Chrome扩展页面的错误信息
